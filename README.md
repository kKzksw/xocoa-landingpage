# XOCOA Landing Page

Static landing page for XOCOA.

## Files

- `index.html` — main collection / landing page
- `manifesto.html` — manifesto page
- `image.png`, `image2.png`, `rum.png` — local visual assets
- `statement1.jpg`, `statement2.jpg`, `statement3.jpg` — manifesto section images
- `LOGO_SOMMELIER_FOND_BLANC_COULEURS.png` — brand logo

## Run locally

You can open the project with VS Code Live Server or any static server.

Example:

```bash
npx serve .
```

## GitHub Pages

This project is ready to be deployed as a static site with GitHub Pages.

Suggested setup:

1. Open the GitHub repository settings
2. Go to `Pages`
3. Set source to `Deploy from a branch`
4. Select branch `main`
5. Select folder `/ (root)`
